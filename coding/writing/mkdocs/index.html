    mkdocs¶

    内置开发服务器，实时预览。

    安装¶

    1. 安装Python和包管理器pip
      1
2
3
4
      $ python --version
Python 2.7.2
$ pip --version
pip 1.5.2
    2. 安装mkdocs
      1
2
3
4
      pip --proxy=http://l0025xxxx:pass%40word@proxy.xxx.com:8080 install mkdocs  # %40代表@

$ mkdocs --version
mkdocs, version 0.15.3

    创建项目¶

    1
2
3
4
    mkdocs new my-project
cd my-project
$ mkdocs serve --dirtyreload  # 启动服务器
$ mkdocs serve -a 127.0.0.1:8001  # 启动第二个服务器，占用8001端口

    serve缓存路径

    C:\Users\xxx\AppData\Local\Temp\mkdocs_xxx

    目录组织方式¶

    1
2
3
    mkdocs.yml  #配置nav菜单，默认所有目录按字母排序

      • 高亮{= = highlight = =}
      • 注释{> > comment < <}
      • 上标H^2^0, text^a\ superscript^
      • 下标CH~3~CH~2~OH, text~a\ subscript~
      • 行内代码高亮：`:::language mycode` or `#!language mycode`

    Admonition¶

    示例，前后要有空行，如果嵌套在列表中，缩进4个空格：

    1
2
3
4
5
6
    !!! type ["custom title or blank"]
    text

# 可折叠，+默认打开
???[+] type ["custom title or blank"]
    text

    摘要，总结

    abstract, summary, tldr

    贴士

    tip, hint, important

    注释，代码片段，说明

    note, snippet, seealso

    举例，列表

    example

    引用，参考链接

    quote, cite

    提示，TODO

    info, todo

    警告

    warning, caution, attention

    危险

    danger, error

    成功，勾选，完成

    success, check, done

    失败

    failure, fail, missing

    问题，疑问，帮助

    question, help, faq

    BUG

    bug
